GitNex 是一款免费/付费的开源 Android 客户端，用于 Git 存储库管理工具 Gitea。 Gitea 是 Gogs 的社区管理分支，Gogs 是用 Go 编写的轻量级代码托管解决方案。

# 功能

- 多账户支持
- 文件及目录浏览器
- 文件查看器
- 创建文件
- 通过令牌登录
- 合并拉取请求
- 离线模式(缓存)
- 探索仓库
- 公开仓库 PR 文件的差异视图
- 仓库列表
- 组织列表
- 创建新仓库/组织
- 搜索/过滤仓库/组织/问题等
- 资料视图
- 问题列表
- 问题评论
- 评论问题
- 创建带多个任务受托人、标签的新问题并向问题添加里程碑、到期日期
- 创建标签
- 编辑/删除标签
- 里程碑列表
- 创建新里程碑
- 分支列表
- 带源码下载的发布
- 仓库的协作者视图
- Markdown 支持
- Emoji 支持
- 扩展设置
- 访问本地非 https 安装的选项
- 2FA OTP 支持
- 通知
- 草稿
- 在文件浏览器内切换分支
- 仓库提交
- 自签名证书支持
- 主题自动切换器
- 删除评论
- 从设置配置应用缓存
