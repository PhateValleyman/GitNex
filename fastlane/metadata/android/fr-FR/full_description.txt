GitNex est un client Android open-source gratuit/payant pour gérer vos projets Git hébergés avec Gitea. Gitea est un projet communautaire dérivé de Gogs, une forge logicielle légère écrite en Go.

# Fonctionnalités

- Comptes multiples
- Explorateur de fichiers
- Aperçu des fichiers
- Création de fichiers
- Autentification par jeton
- Fusions de demandes d’ajout
- Mode hors-ligne (cache)
- Recherche de dépôts
- Aperçu des diff dans les demandes d’ajout
- Liste des dépôts
- Liste des organisations
- Création de dépôts et d’organisations
- Filtres de liste
- Aperçu des profils
- Liste des tickets
- Commentaires de ticket
- Réponse aux commentaires de ticket
- Création de tickets avec assignation multiple, étiquettes, jalons, date d’échéance, etc
- Création d’étiquettes
- Modification/suppression d’étiquettes
- Liste des jalons
- Création de jalons
- Liste des branches
- Versions téléchargeables avec la source
- Aperçu des participants d’un dépôt
- Syntaxe Markdown
- Émoticônes
- Réglages avancés
- Option d’accès local non-HTTPS
- Double authentification OTP
- Notifications
- Brouillons
- Changement de branche depuis l’explorateur de fichiers
- Révisions du dépôt
- Gestion des certificats auto-signés
- Thème jour/nuit
- Suppression de commentaires
- Cache de l’application configurable

Toutes les fonctionnalités : https://codeberg.org/gitnex/GitNex/wiki/Features
