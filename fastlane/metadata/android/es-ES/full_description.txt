GitNex es un cliente nativo Android de pago/gratuito, de código abierto para repositorios Git alojados en Forgejo y Gitea.

# Características

- Soporte para varias cuentas
- Buscador de archivos y directorios
- Visor de archivos
- Crear archivos/incidencias/PRs/wikis/hitos/lanzamientos/etiquetas
- Lista de Pull Requests
- Lista de repositorios
- Lista de organizaciones
- Lista de incidencias
- Lista de etiquetas
- Lista de hitos
- Listas de lanzamientos
- Páginas Wiki
- Modo sin conexión
- Explorar repositorios/incidencias/organizaciones/usuarios
- Vista de perfil
- Soporte para Markdown
- Soporte para emoticonos
- Muy configurable
- Soporte 2FA OTP
- Notificaciones
- Borradores
- Commits del repositorio
- Soporte para certificados autofirmados
- Temas
- y mucho más...
